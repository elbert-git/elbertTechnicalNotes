Javascript

The lingua-franca of full-stack web development. Run on the browsers and web-servers. 

Where does it runs

Javascript runs on browsers via the script that and on servers via environments like node. And because of that the language slightly behaves differently depending on where you run it. 

Miscellaneous notes

Javascript has many quirks due it's quick creation and long history.

ES6 

The modern way of writing javascript that has been adopted widely since 2016 (see below for more details)

Typescript 

A new language that can be easily transpiled for a better developer experience. Removes a good number of weird bugs javascript would have. (at the cost of some boiler plate)
